# \# 🎮 Преодоление: Путь Испытаний

# 

# Игра-платформер с преодолением препятствий. Проект выполнен в рамках курса «Системная и программная инженерия».

# 

# \## 📋 О проекте

# 

# «Преодоление: Путь Испытаний» — динамичная 2D игра, в которой игрок управляет персонажем, преодолевает препятствия (ямы, камни, кусты, движущиеся платформы) и собирает бонусы. Игра развивает реакцию и координацию.

# 

# \### 🎯 Основные особенности

# 

# \- ⚡ Быстрый геймплей (бесконечный раннер)

# \- 🪨 4 типа препятствий с уникальным поведением

# \- 🪙 Система бонусов и рекордов

# \- 💾 Локальное сохранение прогресса (JSON)

# \- 🎮 Простое управление (клавиатура / геймпад)
